## ðŸŽ® About Chessnoth

Chessnoth is a turn-based tactical RPG built on Mantle Network. Players build teams of NFT characters and engage in strategic combat on an 8x8 grid.

**Key Features:**
- Turn-based tactical combat with terrain effects
- 10 unique character classes with distinct abilities
- Fully on-chain NFT characters (ERC721)
- Integrated token economy (CHS - ERC20)
- NFT marketplace for trading characters
- Progressive difficulty with boss battles
